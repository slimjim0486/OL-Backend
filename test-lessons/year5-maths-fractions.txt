Year 5 Mathematics: Adding and Subtracting Fractions

Learning Objective: Add and subtract fractions with the same denominator and denominators that are multiples of the same number.

Key Vocabulary:
- Numerator: the top number in a fraction
- Denominator: the bottom number in a fraction
- Equivalent: fractions that have the same value

Rule 1: Same Denominators
When adding fractions with the same denominator, just add the numerators.

Example:
2/5 + 1/5 = 3/5

The denominator stays the same!

Rule 2: Different Denominators
When denominators are different, find a common denominator first.

Example:
1/2 + 1/4 = ?

Step 1: Find a common denominator (4 works because 2 x 2 = 4)
Step 2: Convert 1/2 to 2/4
Step 3: Add: 2/4 + 1/4 = 3/4

Subtracting Fractions:
The same rules apply for subtraction.

Example:
3/4 - 1/4 = 2/4 = 1/2

Remember to simplify your answer if possible!

Practice Problems:
1. 1/6 + 2/6 = ?
2. 3/8 + 2/8 = ?
3. 1/2 + 1/6 = ?
4. 5/6 - 1/6 = ?
5. 3/4 - 1/2 = ?

Top Tip: Always check if your answer can be simplified to its lowest terms!
